# สูตร Prompt ที่ใช้ได้จริง

ไม่ใช่คำวิเศษ
แต่คือโครงสร้างของ Brief

**Role → Task → Context → Inputs → Rules → Examples → Output → Reminders**

จำแค่นี้ก่อน:
AI เก่งขึ้นทันทีเมื่อคุณบอก “บริบท” ให้ครบ

---

# 1. Role

บอก AI ว่าให้ทำหน้าที่เป็นใคร

อย่าพิมพ์แค่ว่า:
“ช่วยเขียนโพสต์ให้หน่อย”

ให้พิมพ์ว่า:
“คุณคือ Senior Content Strategist ของ Limitless Club ที่ช่วยสื่อสารกับเจ้าของธุรกิจไทย”

Role ทำให้ AI เลือกมุมคิดถูกตั้งแต่ต้น

---

# 2. Task

บอกงานให้ชัดว่าอยากได้ผลลัพธ์อะไร

ไม่ใช่:
“วิเคราะห์อันนี้ให้หน่อย”

แต่เป็น:
“วิเคราะห์ transcript นี้ แล้วสรุปเป็นบทเรียน 5 ข้อสำหรับเจ้าของธุรกิจที่อยากใช้ AI ทำงานดีขึ้น”

งานที่ดีต้องวัดได้ว่า “ตอบครบหรือยัง”

---

# 3. Context

นี่คือส่วนที่คนส่วนใหญ่ลืม

AI ไม่ได้รู้ธุรกิจคุณ
ไม่ได้รู้ลูกค้าคุณ
ไม่ได้รู้เป้าหมายคุณ

คุณต้องบอก:
- ธุรกิจนี้คืออะไร
- กลุ่มเป้าหมายคือใคร
- ใช้ผลลัพธ์นี้ไปทำอะไร
- น้ำเสียงควรเป็นแบบไหน

Context ลดการเดา

---

# 4. Inputs + Rules

แยก “ข้อมูล” ออกจาก “คำสั่ง” ให้ชัด

ใช้โครงแบบนี้:

<context>
บริบทของธุรกิจ / ลูกค้า / เป้าหมาย
</context>

<input>
ข้อมูลจริง เช่น transcript, note, email, sales data
</input>

<rules>
ห้ามเดา / ห้ามแต่งข้อมูล / ถ้าไม่มั่นใจให้บอกว่าไม่มั่นใจ
</rules>

---

# 5. Examples

ถ้าอยากได้งานสไตล์ไหน
ให้ยกตัวอย่าง

AI เรียนรู้จาก pattern ได้ดีมาก

ตัวอย่างที่ควรให้:
- Hook ที่ชอบ
- Caption ที่ใช่
- Output ที่เคยเวิร์ก
- ตัวอย่างสิ่งที่ไม่เอา

ถ้างานมี taste
อย่าปล่อยให้ AI เดา taste เอง

---

# 6. Output Format

ถ้าอยากได้คำตอบที่ใช้ต่อได้ทันที
ต้องบอก format

เช่น:
- 7 slides carousel
- แต่ละ slide มี headline + supporting sentence
- เขียน caption ให้ด้วย
- ปิดท้ายด้วย CTA
- ใช้ภาษาง่าย แต่ดู premium

AI จะทำงานดีขึ้นเมื่อรู้ว่า “ต้องส่งงานหน้าตาแบบไหน”

---

# Prompt Template ที่ผมใช้

คุณคือ [บทบาท]

Task: [งานที่ต้องการ]
Audience: [คนอ่าน / ลูกค้า]
Context: [บริทธุรกิจและเป้าหมาย]
Inputs: [ข้อมูลที่ให้วิเคราะห์]
Rules: [ข้อห้าม / น้ำเสียง / มาตรฐาน]
Examples: [ตัวอย่างที่ชอบ]
Output format: [รูปแบบคำตอบ]
Reminder: [กฎสำคัญที่ต้องย้ำ]

Copy สูตรนี้ไปใช้ได้เลย
