# 🎨 Image Generation Prompts Library

Central archive of every prompt used to generate images for Limitless Club / Jet's brand.
Each render saves its prompt here so we can reuse, remix, and audit.

## Convention

- One markdown file per prompt: `YYYY-MM-DD-<slug>.md`
- Frontmatter must include: `model`, `size`, `quality`, `output_path`, `tags`
- Body = full prompt text exactly as sent to the API
- Group by use case via `tags:` (e.g. `orgchart`, `carousel`, `workshop`, `moodboard`)
