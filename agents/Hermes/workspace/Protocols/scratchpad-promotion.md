# Scratchpad Promotion Rule

Use `Scratchpad/inbox.md` for fast capture.

## Promotion guide
- Delete or leave in place if truly temporary
- Move to `Daily/` if it matters today
- Move to `Protocols/` if it is repeatable
- Move to `../OpenClaw/memory.md` if it becomes durable shared context
- Add to `../Shared Memory/Daily/YYYY-MM-DD.md` if another agent needs it now
