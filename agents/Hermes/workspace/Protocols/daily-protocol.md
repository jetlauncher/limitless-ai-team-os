# Daily Protocol

Use this protocol when updating Hermes day notes.

## Rule
- Keep entries short and scannable
- Log decisions, outcomes, and open loops
- Move cross-agent items into Shared Memory Daily
- Move durable cross-agent items into OpenClaw Memory

## Suggested flow
1. Rough notes in `Scratchpad/inbox.md`
2. Distill into `Daily/YYYY-MM-DD.md`
3. Promote shared items if needed
