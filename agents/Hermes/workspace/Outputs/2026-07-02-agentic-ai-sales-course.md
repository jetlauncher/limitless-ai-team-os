## Overview

A 2-day hands-on bootcamp where every participant builds and deploys a working AI sales agent for their own business. Dual Thai + English tracks. Based on real agent deployments and the Greg Isenberg Agent SaaS playbook.

## What You Build in 2 Days

- Research prospects — scan websites, LinkedIn, databases to find qualified leads
- Qualify inbound leads — ask the right questions, score fit, route to the right person
- Send personalized outreach — multi-channel: email, LINE, WhatsApp, voice calls
- Follow up automatically — timed sequences that match your brand voice
- Book meetings — sync with your calendar, send confirmations, handle reschedules
- Update your CRM — log every interaction, track pipeline, flag hot leads
- Report to you daily — summary of what it did, what worked, what needs your attention

## Day 1: Foundations + Build Your First Agent (9:00 - 17:00 BKK)

### Session 1 (09:00-10:30): The Agentic Sales Landscape
- Why agentic AI is the biggest shift in sales since CRM ($800M ARR proof)
- SaaS vs Agent SaaS: selling work, not software
- The 5 traits of a workflow worth automating
- Market map: Salesforce Agentforce, Explee, Polsia, custom builds
- Where AI excels vs where humans still win

### Session 2 (10:45-12:30): Pick Your Sales Workflow + Shadow the Human
- List 20 sales jobs in your business that repeat daily
- Score each on the 5 traits — pick ONE to automate first
- Shadow exercise: screen-record yourself doing that job 3 times
- Write the 7-part agent spec: trigger, context, tools, actions, approval, escalation, success

### Session 3 (13:30-15:00): Build the Minimum Useful Agent (Live)
- Choose your agent shape: draft-and-approve, triage, coordinator, or bounded action
- Set up the agent using Hermes Agent framework (or no-code tools)
- Connect tools: Gmail/Outlook, LINE, WhatsApp Business, calendar, CRM
- Write the agent's system prompt
- First test: run it on 5 real examples

### Session 4 (15:15-17:00): The Wrapper + Evals
- Build the trust layer: logs, approval dashboard, handoff rules, daily summary
- Create your eval set: 50 real examples with marked right answers
- Run the agent through the eval gym
- Fix the top 3 failure modes
- Homework: let your agent run overnight on real inbound/outbound

---

## Day 2: Deploy, Sell, Scale (9:00 - 17:00 BKK)

### Session 5 (09:00-10:30): Review Overnight Results + Fix
- What did your agent do overnight? Review logs, approvals, failures
- Debug session: fix prompt issues, tool errors, edge cases
- Iterate: re-run evals, confirm improvement
- Upgrade: add a second capability

### Session 6 (10:45-12:30): Voice + Multi-Channel
- Add a voice agent: answer calls, qualify, book (ElevenLabs + Twilio)
- Multi-channel orchestration: email + LINE + voice + SMS together
- Approval workflows: when does the agent act alone vs ask a human?
- Compliance and brand safety: guardrails for outbound, consent, data privacy

### Session 7 (13:30-15:00): Sell It Like Labor — Pricing + Positioning
- How to price: setup fee + monthly, per-outcome, or hybrid
- Pricing models: $1,500 setup + $1,000/mo, $30/qualified meeting, $3K/mo up to 500 tickets
- Outcome pricing: the future of agent SaaS
- Positioning: "We answer your missed calls and book qualified appointments"
- Draft your offer: one sentence, one price, one promise

### Session 8 (15:15-17:00): Distribution + 30-Day Launch Plan
- Workflow teardowns: old way (painful) vs agent way (smooth)
- Pick one platform — make 10 teardown posts in class
- Sales process: DM script, discovery call, pilot agreement, Stripe link
- 30-day plan: week-by-week milestones from pilot to first paying client
- Graduation: present your live agent + offer to the cohort
