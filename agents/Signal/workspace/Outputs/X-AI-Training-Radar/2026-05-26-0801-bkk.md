### 2. OpenAI — Codex Goal Mode
- **Why it matters:** Goal Mode signals a shift from command-by-command prompting to asynchronous delegation. The user defines the goal; the agent decides steps and can continue work over longer spans.

- **Jet content angle:** “Stop prompting. Start delegating.” Convert this into an AI employee SOP framework.
