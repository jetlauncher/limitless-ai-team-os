Core idea from Jet's Local Model/Data Control YouTube transcript:

- Local Model คือ AI ที่รันบนคอมพิวเตอร์ของคุณ
- ไม่ต้องใช้ Cloud
- ไม่ต้องใช้อินเทอร์เน็ต
- สร้างงานได้ด้วยต้นทุน 0 บาท / ฟรี ถ้ามีคอมที่แรงพอ
- LM Studio คือแอปที่ทำให้มีประสบการณ์คล้าย ChatGPT บนเครื่องของเรา
- ตัวอย่างโมเดลในคลิป: Qwen 3.6 35B parameters
- Parameters ยิ่งเยอะ ยิ่งฉลาด แต่ใช้ RAM และพื้นที่มากขึ้น
- ถ้าจะซื้อคอมใหม่สำหรับ Local Model ให้เน้น RAM เยอะที่สุดก่อน
- Cloud AI ยังเหมาะกับงานที่ต้องการรุ่นเก่งที่สุดและความเร็ว
- Local Model เหมาะกับงานที่ต้องการควบคุมข้อมูล ใช้ซ้ำเยอะ และลดต้นทุน
