# Analysis — Local Model Morandi Journal Infographic

## Topic
How Thai founders can decide when to use Local Model vs Cloud AI.

## Audience
Thai SME founders, operators, creators, and business owners who already understand ChatGPT but may not understand local AI models.

## Source language
Thai.

## Output language
Thai.

## Content type
Educational founder guide / product decision guide.

## Complexity
Medium. Needs to explain technical trade-offs without becoming an engineer-only explainer.

## Recommended layout
`dense-modules`

Reason:
- The topic has several decision factors: cost, privacy, hardware, tools, limitations, best use cases.
- A high-density guide gives founders practical decision value in one saved image.

## Selected style
`morandi-journal`

Reason:
- Makes a technical AI topic feel approachable, warm, and saveable.
- Good contrast against previous dark/cinematic comic versions.
- Fits a “founder notebook” / “AI operating manual” vibe.

## Aspect ratio
4:5 vertical, optimized for Instagram/LinkedIn mobile feed.

## Design tone
Warm, hand-drawn, premium educational notebook. Not childish, not corporate.

## Brand adjustments
- Include small header: “Limitless Club”
- Include footer: “@jeditrinupab”
- Keep Morandi palette, but use clean readable Thai typography.

## Risk / QA notes
- Thai text can be error-prone in image models; keep labels short and large enough.
- Avoid overloading with full transcript wording.
- Use fewer but stronger labels rather than tiny paragraphs.
