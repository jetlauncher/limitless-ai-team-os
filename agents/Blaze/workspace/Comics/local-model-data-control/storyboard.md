# Storyboard — ข้อมูลบางอย่างไม่ควรขึ้น Cloud

Source: Jet transcript — “คู่มือใช้ Local Model เพื่อใช้ AI ฟรี ไม่ต้องจ่าย”

## Page Setup

- Page count: 1
- Layout: strict 2×2 four-panel grid
- Aspect ratio: 4:3 landscape
- Art style: minimalist premium black-and-white line art with bronze/gold spot color `#94764A`
- Tone: clever, practical, strategic

## Title

ข้อมูลบางอย่างไม่ควรขึ้น Cloud

## Core takeaway

Local Model ไม่ใช่แค่เรื่องใช้ฟรี แต่คือเรื่อง “การควบคุมข้อมูล”

## Four-Panel Structure

### Panel 1 — Setup
Founder holds a folder of sensitive business data and is about to drag it into a big Cloud AI.

Dialogue: “Local Model ไม่ใช่แค่ AI ฟรี…”

### Panel 2 — Development
A warning sign appears: “ข้อมูลนี้ควรอยู่ที่ไหน?” The folder contains customer, sales, and new idea labels.

Dialogue: “แต่คือคำถามว่า ข้อมูลนี้ควรอยู่ที่ไหน?”

### Panel 3 — Turn
A split road appears: Cloud AI for public/general work, Local Model inside a locked laptop/vault for private data. Strong bronze accent on the local lock.

Labels: “Public → Cloud” and “Private → Local”

### Panel 4 — Conclusion
Founder now stands in front of a simple decision dashboard with four factors: capability, cost, privacy, control.

Dialogue: “AI Strategy = ความเก่ง + ต้นทุน + ความเป็นส่วนตัว + การควบคุม”

Caption: “Cloud เพื่อความเร็ว Local เพื่อการควบคุม”
