# Characters — Local Model Data Control

Preset: four-panel minimalist
Language: Thai primary captions; English prompt instructions

## Founder
- Simplified Thai founder/operator stick-figure, round glasses, black T-shirt, laptop.
- Curious but almost careless with data in panel 1; more strategic by panel 4.

## Cloud AI
- Large friendly cloud icon with plug/cable, labeled “Cloud AI”.
- Bright and useful, but visually outside the company wall.

## Local Model
- Small glowing AI brain inside a locked laptop/vault, labeled “Local Model”.
- Bronze/gold accent on the lock and secure boundary.

## Sensitive Data Folder
- Simple folder labeled “ลูกค้า / ยอดขาย / ไอเดียใหม่”.
- Should be the recurring object across panels.

## Palette
- Mostly black-and-white line art.
- Spot accent color: bronze/gold `#94764A` only for privacy/control elements, split-path rule, and final takeaway.
