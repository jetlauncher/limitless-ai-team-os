# Characters — AI Agent Team Creative Director

Preset: four-panel minimalist
Language: Thai primary captions; English prompt instructions

## Founder
- Archetype: Thai SME founder / creator-operator
- Visual cues: simplified stick-figure character, round glasses, black T-shirt, small laptop, slightly messy desk in panel 1
- Personality: ambitious but overloaded; wants leverage without losing control

## AI Agent Team
- Archetype: tiny specialist operators inside a command room
- Visual cues: three small stick-figure mini-agents at tiny workstations, each with a simple label
  - Signal: tiny radar screen
  - Blaze: tiny content board / pencil
  - Higgsfield: tiny image/video frame
- Personality: precise, productive, not chaotic

## Approval Gate
- Archetype: system guardrail, not a person
- Visual cues: bronze/gold checkpoint gate or button labeled “Approve?”
- Role: prevents AI output from becoming public without founder decision

## Symbol Palette
- Mostly black-and-white line art
- Spot accent color: bronze/gold `#94764A`, matching Limitless Club
- Accent only for: agent labels, approval gate, clean dashboard, final key takeaway
