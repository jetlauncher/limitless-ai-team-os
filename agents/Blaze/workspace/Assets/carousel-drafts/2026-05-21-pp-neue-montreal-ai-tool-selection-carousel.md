# Carousel Draft — PP Neue Montreal Theme

Date: 2026-05-21
Theme reference: PP Neue Montreal / dark founder editorial / creative edge
Language: Thai primary
Status: Text draft for approval before rendering

## Visual direction

- Format: Instagram 4:5 carousel, 1080x1350
- Base: deep black / charcoal background
- Typography direction: PP Neue Montreal for English words, numbers, section labels, UI-style metadata; Thai-safe sans such as Sukhumvit Set or LINE Seed Sans TH for Thai body/headlines
- Mood: founder operating system, clean Swiss grid, slightly rebellious creative edge
- Palette: black, graphite, warm white, silver, muted bronze accent
- Layout: big left-aligned headlines, small metadata labels, gridlines, tiny system marks, high negative space
- Header: LIMITLESS CLUB
- Footer: @jeditrinupab

## Caption angle

คนที่ชนะด้วย AI ไม่ใช่คนที่มี tool เยอะที่สุด แต่คือคนที่รู้ว่า tool ไหนควรอยู่ตรงไหนในระบบธุรกิจ

## Slide copy

### Slide 1 — Hook
Eyebrow: AI OPERATING SYSTEM
Headline:
AI เยอะ
ไม่ใช่ปัญหา
ถ้าคุณมีระบบเลือก
Body: ปัญหาของเจ้าของธุรกิจวันนี้ไม่ใช่ “ไม่มี AI ใช้” แต่คือ “ไม่รู้ว่าอะไรควรใช้กับงานไหน”
CTA: Save this before buying another tool

### Slide 2 — Problem
Eyebrow: COMMON MISTAKE
Headline:
อย่าเริ่มจาก
“ตัวไหนดีสุด?”
Body: เพราะ AI แต่ละตัวไม่ได้ชนะทุกสนาม บางตัวเก่งคิด บางตัวเก่งภาพ บางตัวเก่งลงมือทำ
CTA: เริ่มจากงาน ไม่ใช่ชื่อ tool

### Slide 3 — Reframe
Eyebrow: THE BETTER QUESTION
Headline:
ถามใหม่ว่า
“งานนี้ต้องการ
สมองแบบไหน?”
Body: ถ้าคุณถามถูก คุณจะหยุดไล่ตามทุกกระแส แล้วเริ่มจัดทีม AI ให้ทำงานแทนคุณได้จริง
CTA: Question creates leverage

### Slide 4 — Founder Stack
Eyebrow: MINIMUM STACK
Headline:
ธุรกิจเล็ก
ไม่ต้องมี AI 20 ตัว
Body: เริ่มจาก 3 หมวดพอ: ตัวคิด / ตัวสร้าง / ตัวลงมือทำ
CTA: Think. Create. Execute.

### Slide 5 — Tool Roles
Eyebrow: ROLE MAP
Headline:
1. ChatGPT = คิดและจัดระบบ
2. Creative AI = สร้างภาพ/วิดีโอ
3. Agent = ลงมือทำงาน
Body: เมื่อแต่ละตัวมีหน้าที่ชัด ทีมของคุณจะ lean ขึ้น โดยไม่วุ่นวายขึ้น
CTA: Tools need job descriptions too

### Slide 6 — Founder Insight
Eyebrow: REAL ADVANTAGE
Headline:
AI ไม่ได้แทนทีม
แต่มันแทน
“งานที่ทีมไม่ควรเสียเวลา”
Body: เจ้าของธุรกิจควรใช้ AI เพื่อลดงานซ้ำ เพิ่ม speed และปล่อยให้คนจริงไปทำงานที่ต้องใช้ judgement
CTA: Automate the drag, not the trust

### Slide 7 — CTA
Eyebrow: LIMITLESS CLUB
Headline:
ถ้าคุณมี AI เยอะ
แต่ยังไม่เร็วขึ้น
แปลว่าคุณยังขาดระบบ
Body: เริ่มจากเลือก 3 งานที่กินเวลาคุณที่สุด แล้วหา AI มาช่วยตรงนั้นก่อน
CTA: Comment “STACK” ถ้าอยากได้ AI stack สำหรับเจ้าของธุรกิจ
