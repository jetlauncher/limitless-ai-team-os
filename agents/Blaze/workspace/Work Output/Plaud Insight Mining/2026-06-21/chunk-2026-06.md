### Main
การบรรยายนี้ครอบคลุมการใช้งาน AI Tools สำหรับธุรกิจอย่างครบถ้วน ตั้งแต่แพลตฟอร์ม Workflow Automation อย่าง N8N และ Zapier เครื่องมืออัดเสียง PLAB การสร้าง AI Employee ด้วย Claude Projects และ Custom GPT ไปจนถึงการใช้ ChatGPT Agent Mode, Deep Research, SEEDANCE 2.0 สำหรับสร้างวิดีโอ และแนวคิด AI OS เพื่อให้ AI รู้จักธุรกิจของเราอย่างลึกซึ้ง เพื่อเพิ่ม Leverage และประหยัดเวลาในการทำงาน
### Signals
- N8N และ Zapier - แพลตฟอร์ม Workflow Automation: N8N และ Zapier เป็นแพลตฟอร์มประเภท Workflow Automation ที่ถูกออกแบบมาเพื่อเชื่อมต่อแพลตฟอร์มหนึ่งกับอีกแพลตฟอร์มหนึ่ง เช่น Microsoft Excel Sync กับ Gmail โดย Zapier มีมาก่อน N8N ประมาณ 4-5 ปี และถือว่าเป็น Pioneer ในสายงานนี้
- PLAB - เครื่องมืออัดเสียงและแปลงเป็น Text: PLAB (P.L.A.B.) เป็นเครื่องมืออัดเสียงที่มี AI ในตัว สามารถแปลงเสียงพูดเป็น Text ได้ จากนั้นนำ Text ไปใช้งานกับ AI ตัวอื่นต่อได้ทันที ราคาประมาณ 6,000 บาท และสามารถสร้างมูลค่าให้ธุรกิจได้อย่างน้อย 2-3 ล้านบาท
- ChatGPT Codex - AI Agent สำหรับงาน Developer: Codex เป็นแอปพลิเคชันของ ChatGPT ที่ใช้สำหรับ AI Agent คาดว่าจะถูก Merge รวมกับ ChatGPT ภายใน 1-2 เดือน (ประมาณ กรกฎาคม-สิงหาคม 2026) โดยผู้ที่มี ChatGPT Plus ราคา 20 เหรียญต่อเดือนอยู่แล้ว สามารถดาวน์โหลด Codex มาใช้ได้เลยโดยไม่เสียค่าใช้จ่ายเพิ่มเติม
- Skill - SOP สำหรับ AI: Skill คือ Workflow ที่ตั้งค่าล่วงหน้าให้กับ AI เพื่อให้ทำงานซ้ำๆ ในลักษณะเดิมได้อัตโนมัติ เปรียบเสมือน SOP (Standard Operating Procedure) ของโรงงาน ตัวอย่างเช่น Skill ที่แปลงไฟล์เสียงดิบเป็นไฟล์ HTML สรุปใจความ
- Knowledge Base ส่วนตัว - ฐานความรู้จากสิ่งที่เคยพูดและสอน: Knowledge Base ส่วนตัวคือเว็บไซต์หรือฐานข้อมูลที่รวบรวมทุกสิ่งที่เคยพูดหรือสอนไว้ สามารถค้นหาด้วย Keyword แล้วกรองไฟล์ที่เกี่ยวข้องออกมาได้ทันที เพื่อนำไปสร้างเนื้อหาต่อในรูปแบบต่างๆ
- Slack - แพลตฟอร์มสื่อสารองค์กรที่เชื่อมต่อกับ AI: Slack เป็นแอปพลิเคชันสื่อสารในองค์กร คล้าย LINE แต่มีความสามารถด้าน Automation และการเชื่อมต่อกับ AI ที่ดีกว่ามาก มี Permission Management ที่ละเอียดกว่า และสามารถแจ้งเตือน Real-time เมื่อมีการเปลี่ยนแปลงใน Google Sheets หรือ Excel ได้
- Mac vs Windows - การเลือกเครื่องสำหรับยุค AI: Mac มีความปลอดภัยสูงกว่า Windows เพราะทำงานบนพื้นฐาน Linux ซึ่งมีความปลอดภัยสูงกว่า Windows ที่มีคนพยายาม Hack ตลอดเวลา นักพัฒนา AI รายใหญ่ทั่วโลก เช่น ChatGPT, Anthropic, Google ล้วนใช้ Mac ในการ Demo บน Stage
- การจัดการไฟล์ด้วย iCloud และ Google Drive: การแบ่งการจัดเก็บไฟล์ระหว่าง iCloud และ Google Drive โดย iCloud เหมาะสำหรับไฟล์งานหลังบ้านและไฟล์ส่วนตัวที่ sync อัตโนมัติบน Mac ส่วน Google Drive เหมาะสำหรับงานหน้าบ้านที่ต้องแชร์กัน
- Airtable คืออะไรและทำไมถึงใช้แทน Excel: Airtable เป็นเครื่องมือจัดการฐานข้อมูลที่คล้ายกับ Microsoft Excel แต่ทรงพลังกว่า โดยสามารถทำงานร่วมกับ AI ได้ทุกตัว รองรับหลาย Sheet, หลาย View ในหนึ่ง Sheet, สร้างฟอร์ม, สร้าง Dashboard (Interface), ทำ Automation และเชื่อมต่อกับแพลตฟอร์มภายนอกได้
- ChatGPT Real Time Voice - การแปลภาษาแบบเรียลไทม์: ChatGPT ได้ออก Model ใหม่ชื่อว่า Real Time Voice ซึ่งสามารถแปลเสียงพูดเป็นภาษาต่างๆ ได้แบบ Real Time รองรับมากกว่า 70 ภาษา เหมาะสำหรับการสื่อสารกับลูกค้าต่างชาติ
- AI Voice สำหรับธุรกิจ - Next Phase: การนำ AI Voice มาใช้ในธุรกิจเป็น Next Phase ถัดจากการพิมพ์ โดย AI Voice สามารถเชื่อมต่อกับ CRM, เว็บไซต์, อีเมล, TikTok, Shopee และระบบต่างๆ เพื่อช่วยลูกค้าตอบคำถาม เช็คออเดอร์ และจัดการงานต่างๆ ได้อัตโนมัติ
- ChatGPT Image 2.0 และการสร้างคลิปวิดีโอด้วย AI: ChatGPT Image 2.0 เป็นโมเดลสร้างรูปภาพล่าสุดที่เก่งที่สุดในตลาด โดยเฉพาะภาษาไทย ความละเอียดสูง และ Realistic สามารถนำ Storyboard มาสร้างเป็นคลิปวิดีโอได้ภายในไม่ถึง 1 ชั่วโมง จากเดิมที่ใช้เวลา 3-4 ชั่วโมงต่อคลิป 30-45 วินาที

## 2026-06-06_การใช้-ai-agent-และ-codex-สำหรับระบบปฏิบัติการทำงานอัตโนมัต.md
Title: ---
### Main
การบรรยายนี้มุ่งเน้นการประยุกต์ใช้ AI Agent และแอปพลิเคชัน Codex เพื่อสร้างระบบปฏิบัติการอัตโนมัติ (AIOS) สำหรับการทำงาน ครอบคลุมการใช้ Skills และเครื่องมือต่างๆ เช่น Remotion, Clipify, การตั้ง Schedule Task, และเชื่อมต่อ API ภายนอกอย่าง Apify รวมไปถึงการประยุกต์ใช้ในการวิเคราะห์คู่แข่ง การตัดต่อวิดีโออัตโนมัติ และระบบช่วยจัดการงาน
### Signals
- การใช้ AI ในกระบวนการตัดต่อวิดีโออัตโนมัติ: การประยุกต์ใช้ AI Skill ได้แก่ Remotion และ Clipify ที่ช่วยจับเสียง วิเคราะห์เนื้อหา และเพิ่ม B-Roll/Overlay เข้าไปในคลิป จากนั้นนำมาตัดเป็นคลิปสั้นรูปแบบแนวตั้งพร้อมแคปชั่น
- การใช้งาน Schedule Task สำหรับ AI Agent: การตั้งเวลาการทำงาน (Automation) ให้ AI Agent ทำงานต่างๆ เช่น การดึงข้อมูลหรือการทำงานวิจัย ในช่วงเวลาที่ไม่ได้ใช้คอมพิวเตอร์เพื่อประหยัดเวลา
- Whisper Flow: เครื่องมือ Speech to Text ที่เปลี่ยนคำพูดเป็น Prompt โดยมี AI อยู่หลังบ้าน ช่วยหาข้อมูลและทำ PDF Report ได้ทั้งภาษาไทยและภาษาอังกฤษ
- การใช้ Telegram ร่วมกับ AI Agent: แพลตฟอร์มที่เหมาะสมที่สุดในการสร้างทีม AI Agent 10 กว่าตัว เนื่องจากเชื่อมต่อง่ายและทำงานแบบ Proactive ได้ดีกว่าแพลตฟอร์มอื่น
- แอปพลิเคชัน Codex: ซอฟต์แวร์ AI ที่ติดตั้งและทำงานในระดับ Local บนคอมพิวเตอร์ เปรียบเสมือนการนำสมอง AI มาไว้ในเครื่องตัวเอง
- การรักษาความปลอดภัยของข้อมูลด้วย Local AI: วิธีการป้องกันข้อมูลความลับหรือข้อมูลที่เตรียมทำ Patent โดยใช้โมเดล AI ที่รันบนคอมพิวเตอร์ส่วนตัว (Local AI) เพื่อป้องกันข้อมูลรั่วไหลสู่อินเตอร์เน็ต
- Agentic Workflow และ AI Operating System: การสร้างระบบปฏิบัติการ AI ที่เป็นศูนย์กลางของธุรกิจ โดยการใช้ AI Agent ซึ่งสามารถทำงานจบในตัวเอง มีการกำหนดเป้าหมาย เครื่องมือ และชุดข้อมูลเฉพาะสำหรับแต่ละตำแหน่งงาน
- กลุ่มเครื่องมือ AI Agent ทั้ง 3 ประเภท: การแบ่งประเภทเครื่องมือ AI Agent ในปัจจุบันออกเป็น 3 กลุ่มตามความเหมาะสมในการใช้งาน ราคา และความยืดหยุ่นในการปรับแต่งระบบ
- การเปรียบเทียบ Core Work และ Codex: การวิเคราะห์จุดเด่นและข้อจำกัดของเครื่องมือ AI สองตัว ได้แก่ Core Work และ Codex เพื่อเลือกใช้ให้เหมาะสมกับลักษณะงาน
- การตั้งค่า AI Operating System (AIOS): การจัดโครงสร้างโฟลเดอร์ในคอมพิวเตอร์เพื่อเป็นระบบปฏิบัติการสำหรับการทำงานร่วมกับ AI Agent ตามตำแหน่งหน้าที่หรือฟังก์ชันขององค์กร
- การตั้งค่าและการใช้งาน AI Agent (Codex และ Core Work): การใช้ AI Agent เพื่อช่วยสร้างเอกสาร จัดระเบียบไฟล์ รวมถึงการตั้งค่าไฟล์ agent.md และระบบ Memory เพื่อให้ AI เข้าใจบริบทและทำงานอัตโนมัติได้แม่นยำขึ้น
- การประยุกต์ใช้ Plugins และเครื่องมือเสริม: การผสานการทำงานระหว่าง AI กับแพลตฟอร์มภายนอกต่างๆ เพื่อขยายขีดความสามารถ ทั้งด้านการเงิน การพัฒนาเว็บไซต์ การจัดการซอร์สโค้ด และฐานข้อมูล

## 2026-06-06_การใช้งาน-ai-agent-codex-และ-manus-และ-automation-workflow.md
Title: ---
### Main
การบรรยายนี้มุ่งเน้นไปที่การประยุกต์ใช้ AI Agent เช่น Codex และ Manus เพื่อเพิ่มประสิทธิภาพการทำงานและสร้างระบบอัตโนมัติ (Automation Workflow) ครอบคลุมถึงการใช้ MCP Servers เชื่อมต่อระบบต่างๆ การทำการตลาดอัตโนมัติ การจัดการ Sales Report, Lead Scoring, Lead Nurturing และการเชื่อมต่อ AI กับอุปกรณ์มือถือ นอกจากนี้ยังให้แนวคิดการประเมินมูลค่าเวลาเพื่อใช้ AI ทำงานแทน และการเพิ่ม Productivity โดยรวม
### Signals
- MCP Servers: MCP เป็นเพียงคำที่ใช้แทนคำว่า API ในบริบทของ AI ทำหน้าที่เป็นจุดเชื่อมต่อให้ Codex สามารถเข้าถึงเครื่องมือและระบบอื่นๆ เช่น ห้องครัว, Cashier, ตารางงาน, อีเมล หรือข้อมูล Spreadsheet ต่างๆ เพื่อนำข้อมูลกลับมาประมวลผล
- Infinite Content Engine: ระบบการตลาดอัตโนมัติที่เกิดจากการทำงานร่วมกันของเครื่องมือต่างๆ ได้แก่ Codex (ศูนย์กลาง), Higgsfield (สร้างคอนเทนต์), Brotato (โพสต์ลงโซเชียลมีเดีย) และ Meta MCP (วิเคราะห์โฆษณา) แบบวนลูปครบวงจร
- การทำงานอัตโนมัติและการทำการตลาดด้วย AI: การใช้เครื่องมือ AI และระบบ Automation ในการทำการตลาดและการทำงานเพื่อเพิ่มประสิทธิภาพ โดยให้ระบบทำงานแทนในบางขั้นตอน
- การจัดการ Sales Report และ Lead Scoring ด้วย AI: การเปลี่ยนวิธีการส่งรายงานการขายเพื่อเพิ่มประสิทธิภาพในการเชื่อมต่อกับ AI และการใช้ AI ในการประเมินศักยภาพของลูกค้าผ่านการทำ Lead Scoring
- การใช้งาน Codex สำหรับ Lead Nurturing และ Enrich Data: การใช้ Plugin Sales ของ Codex เพื่อค้นหา ตรวจสอบ และวิเคราะห์ข้อมูลลูกค้าแบบเจาะลึก เพื่อช่วยให้ทีมขายทำงานได้เร็วและแม่นยำขึ้น
- การสร้างระบบอัตโนมัติ (Schedule Task Automation): การกำหนดให้ Codex ทำงานตามขั้นตอนที่วางไว้เป็นประจำทุกวันหรือทุกสัปดาห์ เพื่อประหยัดเวลาและเพิ่มประสิทธิภาพในการทำงาน
- Mobile Integration การเชื่อมต่อ Codex กับโทรศัพท์มือถือ: การเชื่อมต่อบัญชี Codex เข้ากับแอปพลิเคชัน ChatGPT บนโทรศัพท์มือถือเพื่อการใช้งานที่ต่อเนื่อง
- การตั้งค่าและการเชื่อมต่อแอป Codex: กระบวนการและเงื่อนไขในการเชื่อมต่อแอป Codex ระหว่างคอมพิวเตอร์และอุปกรณ์มือถือ รวมถึงข้อควรระวังในการเลือกสถานที่ติดตั้งเซิร์ฟเวอร์
- การใช้เครื่องมือ Manus (AI Agent): Manus คือ AI Agent บน Cloud ที่มีฟีเจอร์คล้าย Codex แต่มีความสามารถในการสร้างคอมพิวเตอร์เสมือน (Virtual Computer) ของตัวเองในการทำงาน โดยสามารถเชื่อมต่อแพลตฟอร์มต่างๆ และรับคำสั่งทำงานเป็นเลขาหรือผู้ช่วยจัดการงานแทนได้

- ตัวอย่าง CRM Automation ด้วย Codex และ Line OA: [Speaker 2] ยกตัวอย่าง workflow ที่ใช้จริงทุกวัน คือให้ Codex เข้า Line OA ด้วย Browser Use อ่านข้อความลูกค้า เติมข้อมูลลง Excel File ที่เป็น CRM ระบุว่าลูกค้าคนใดทักมา และสร้าง draft follow up สำหรับลูกค้าที่ทีมยังไม่ได้ตอบหรือทักมาเกิน 48 ชั่วโมง จากนั้นเมื่อ [Speaker 2] อ่านและอนุมัติแล้ว Codex จะเข้า Line OA ไปส่ง message หาลูกค้าเองตามที่ตั้งไว้
- Codex แตกต่างจาก ChatGPT แบบ Cloud อย่างไร: Codex ถูกอธิบายว่าเป็นเหมือน “พี่ชายของ ChatGPT” โดยมีความแตกต่างสำคัญ 1 อย่าง คือ ChatGPT ใช้ผ่าน Cloud เช่น เปิด Web App, โหลดแอปในมือถือ หรือโหลดแอปในคอม ส่วน Codex เป็นแอปพลิเคชันที่ต้องดาวน์โหลดลงมาบนคอมพิวเตอร์ของผู้ใช้ ทำให้สามารถทำงานกับไฟล์ในเครื่องได้โดยตรง
- การทำงานกับไฟล์ใหญ่และข้อมูลหลายแหล่งโดยไม่ต้องอัปโหลดขึ้น Cloud: Codex ช่วยให้ AI อ่านไฟล์ที่อยู่ในเครื่องของผู้ใช้ได้ละเอียดและถูกต้องขึ้น โดยไม่ต้องอัปโหลดไฟล์ขึ้น Cloud ซึ่งมีปัญหาในกรณีไฟล์ใหญ่ เช่น Excel ใหญ่, File Project ใหญ่, ไฟล์ vendor, ไฟล์ supplier หรือไฟล์สเปก เพราะเมื่ออัปโหลดไฟล์ใหญ่ขึ้น Cloud แล้ว AI อาจอ่านได้ไม่ครบถ้วน
- Browser Use และ Computer Use ของ Codex: Browser Use หรือ Computer Use คือฟีเจอร์ที่ทำให้ Codex ใช้เว็บบราวเซอร์หรือคอมพิวเตอร์ของผู้ใช้ได้เหมือนมีผู้ช่วยมานั่งหน้าคอมแทน สามารถเข้าเว็บระบบภายใน เปิดแอป กดเมนู ดาวน์โหลดข้อมูล ฟิลเตอร์ข้อมูล และสร้างไฟล์ผลลัพธ์ได้
- ความปลอดภัยและการเข้าถึงข้อมูลในระบบภายใน เช่น Mango ERP: ข้อกังวลเรื่องข้อมูลใน Mango ERP ซึ่งเป็นข้อมูลลับ ถูกอธิบายว่า Codex ทำงานเหมือนผู้ใช้ล็อกอินและกดดาวน์โหลดเองผ่าน Username กับ password ของผู้ใช้ ไม่ใช่การปล่อยให้ระบบดึงอะไรก็ได้โดยไม่มีสิทธิ์ และไม่ได้อัปโหลดข้อมูลออกไปที่อื่น แต่ดาวน์โหลดลงคอมพิวเตอร์ของผู้ใช้
