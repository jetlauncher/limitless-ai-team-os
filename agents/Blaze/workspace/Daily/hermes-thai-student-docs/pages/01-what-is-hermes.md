# 01 — Hermes Agent คืออะไร

Hermes Agent คือ AI Agent ที่เรา “สั่งงานให้ทำงานจริง” ได้ ไม่ใช่แค่ chatbot ที่ตอบคำถาม

พูดง่ายๆ:

- ChatGPT ทั่วไป = คุย / ถาม / ให้ช่วยเขียน
- Hermes Agent = คุย + ใช้เครื่องมือ + จำบริบท + ทำงานข้าม session + ตั้งงานอัตโนมัติได้

## Hermes ทำอะไรได้บ้าง

ตัวอย่างงานที่ Hermes ทำได้:

- ค้นเว็บและสรุปข้อมูล
- อ่าน/แก้ไฟล์ในเครื่อง
- รัน command ใน terminal
- เปิด browser แล้วทำงานบนเว็บ
- จำ preference ของผู้ใช้
- สร้างและใช้ “skills” สำหรับ workflow ซ้ำๆ
- ตั้งงานประจำ เช่น สรุปข่าวทุกเช้า
- คุยผ่าน Telegram / Discord / Slack ได้

## คิดภาพง่ายๆ

Hermes คือ “AI intern ที่มีเครื่องมือ”

แต่ต้องสอนให้ชัด:

> ถ้าสั่งไม่ชัด ผลลัพธ์จะมั่วได้  
> ถ้าให้ context ดี ผลลัพธ์จะดีขึ้นมาก

## คำสำคัญที่ควรรู้

| คำ | ความหมายแบบง่าย |
|---|---|
| Agent | AI ที่ทำงานหลายขั้นตอนเองได้ |
| Tool | ความสามารถพิเศษ เช่น เปิดเว็บ อ่านไฟล์ รัน terminal |
| Memory | ความจำระยะยาวของ agent |
| Skill | คู่มือ/ขั้นตอนที่ agent โหลดมาใช้เมื่อเจองานเฉพาะ |
| Gateway | ระบบที่ทำให้คุยกับ Hermes ผ่าน Telegram/Discord/Slack ได้ |
| Cron | การตั้งงานให้ทำอัตโนมัติตามเวลา |

## เหมาะกับใคร

- นักเรียนที่อยากเข้าใจ AI Agent จริงๆ
- คนที่อยากใช้ AI ทำงาน operational ไม่ใช่แค่ถามตอบ
- ผู้ประกอบการที่อยากสร้าง workflow ส่วนตัว
- developer ที่อยากให้ AI ช่วยงานในเครื่อง/บน server
