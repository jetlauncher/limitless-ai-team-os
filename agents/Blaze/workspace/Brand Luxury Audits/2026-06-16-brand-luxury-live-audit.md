# Brand Luxury 1% Live Audit — 2026-06-16

**Focus surface:** YouTube — `youtube.com/@jeditrinupab`  
**Live access:** Successful via public YouTube channel page.  
**Audit scope:** Channel homepage first impression + Videos tab / recent upload packaging.  
**Do not implement without Jet approval:** All recommendations below are draft brand/profile/content changes only.

## What was visible

- Channel banner uses a black base with event/group photos on both sides, Jet speaking in the middle, large Thai line: “คนไทย 1,000,000 คนต้องเก่ง AI”, Limitless Club logo, red Subscribe button graphic, social icons, and `@JEDITRINUPAB`.
- Avatar is a clean dark portrait of Jet; premium, simple, and consistent with founder-media positioning.
- Homepage first row recent thumbnails are high-energy AI tutorial/news packaging: strong face presence, bold Thai/English headline text, saturated colors, icon/tool screenshots, lightning/comparison elements, and occasional small signature mark.
- Recent titles mix Thai and English/tech terms. Some series markers exist (`[Full Course]`, review/tutorial language), but no visible consistent luxury/premium metadata system across thumbnails.
- The strongest premium cue is the avatar + black banner foundation. The weakest premium cue is the high-saturation, high-density thumbnail row fighting against the expensive-looking channel header.

## Micro-upgrades

### 1) Replace the red “SUBSCRIBE” button graphic inside the banner with a restrained bronze/ivory CTA strip
- **Exact change:** In the banner artwork, remove the bright YouTube-red subscribe button and notification bell graphic. Replace it with a small lower-right editorial CTA: `LIMITLESS CLUB` / `AI OPERATING SYSTEM FOR FOUNDERS` / `@jeditrinupab`, using warm ivory text and one rare bronze line or dot.
- **Why it increases luxury association:** Red subscribe graphics feel creator-growth / mass-market. A restrained editorial CTA makes the channel feel like a premium media property, not a YouTube hustle page.
- **Quick implementation note:** Keep the existing banner structure if needed, but redesign only the right CTA module. Use #17181A, #E2D7C8, and #94764A only. No bell icon, no red, no social icon row.
- **Priority:** High.

### 2) Add a consistent “series label rail” to future thumbnails
- **Exact change:** Add a small top-left or bottom-left label on every new thumbnail, e.g. `AI OPERATING SYSTEM`, `FULL COURSE`, `AI TOOL REVIEW`, or `AGENT BUILD`. Same position, same type size, same bronze/ivory treatment.
- **Why it increases luxury association:** Premium brands create memory through repetition. A quiet label rail turns individual videos into an editorial system and makes the channel feel curated rather than improvised.
- **Quick implementation note:** Use a 6–8% height label band, #17181A background, #E2D7C8 uppercase text, optional 1px bronze divider. Keep the main YouTube hook big, but let the label be disciplined.
- **Priority:** High.

### 3) Reduce thumbnail accent palette to one “hero color + bronze” per video
- **Exact change:** For future thumbnails, avoid combining multiple high-saturation accents in one frame. Example: the `AI CLASH` thumbnail uses orange + blue + lightning + split-screen + big face. Keep the split-screen idea, but restrict each thumbnail to: black/graphite base, ivory headline, one tool-color cue, and one bronze highlight.
- **Why it increases luxury association:** Luxury visuals are confident enough to use fewer colors. Restrained color control makes the content feel more expensive while preserving click clarity.
- **Quick implementation note:** Create a thumbnail export checklist: max 1 saturated tool color, max 1 bronze accent, no more than 2 type sizes, no glow unless it supports legibility.
- **Priority:** Medium-high.

### 4) Make title prefixes more editorial and consistent
- **Exact change:** Standardize high-value formats such as:
  - `Full Course:` instead of `[Full Course]` for cleaner typography in title feeds.
  - `Review:` for tool/agent evaluations.
  - `Build:` for implementation/tutorial videos.
  - `Brief:` for trend/news/commentary videos.
- **Why it increases luxury association:** Brackets feel tactical and YouTube-native. Editorial prefixes feel like a premium publication with departments.
- **Quick implementation note:** Apply to future uploads first; optionally update only top evergreen videos after approval. Thai titles can still lead, but the prefix system should be visually consistent.
- **Priority:** Medium.

### 5) Clean the channel description preview into a sharper premium promise
- **Exact change:** Replace the visible description preview from a direct training/Line OA line into a more elevated first line draft, e.g. `สร้าง AI Operating System สำหรับผู้ก่อตั้งและทีมที่ต้องการทำงานระดับ 10x` followed by Line OA link after the first sentence.
- **Why it increases luxury association:** The first visible sentence currently reads like a sales/check-round CTA. A premium promise first, CTA second, improves authority and first impression.
- **Quick implementation note:** Draft only; do not edit live without approval. Keep the Line OA, but move it after the positioning sentence.
- **Priority:** Medium-high.

## Today’s recommended 1% upgrade

**Win:** Redesign only the banner’s right-side CTA module: remove the red subscribe button/bell/social icon row and replace it with a quiet Limitless Club editorial CTA in ivory/bronze.

**Why this is the best 1% move:**
- It is small and contained: one banner module, not a full channel rebrand.
- It immediately removes the least luxurious visible element above the fold.
- It aligns the first impression with Midnight Luxe Editorial while preserving Jet’s authority/photo/social proof context.
