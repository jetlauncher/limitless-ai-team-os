# Brand Luxury 1% Live Audit — Instagram — 2026-06-17

## Scope
- Focus surface: Instagram `instagram.com/jeditrinupab`
- Rotation rationale: yesterday’s audit was YouTube, so today is Instagram.
- Live access: worked in public logged-out view; Instagram blocked deeper post/caption viewing behind login/sign-up prompt after interaction, so this audit is based on the visible public profile, bio, highlights, and first grid impression.
- No public assets were edited, posted, scheduled, boosted, or changed.

## First-impression read
Jet’s Instagram already has strong authority signals: verified badge, 92.7K followers, Forbes proof, 1,000+ students, 3,000M+ sales experience, professional portrait, and visible client/class/event proof. The main luxury gap is not credibility — it is visual restraint and brand memory. The profile currently mixes high-status proof with a more creator/AI-content look: emoji-heavy bio, mixed highlight styles, loud Reel covers, robot stickers, yellow text, event photos, lifestyle images, and editorial quote cards. Premium association would improve fastest by reducing decorative noise and giving the profile a repeatable “Limitless Club editorial system.”

## Micro-upgrades

### 1) Bio detox: convert emoji proof stack into editorial proof stack
- **Exact change:** Draft a cleaner bio for Jet approval that uses fewer emojis and more institution-like wording.
  - Current visible tone: `🚀 AI Coach... / 🏆 ลูกศิษย์... / 🔥 Forbes... / 💰 ประสบการณ์...`
  - Suggested draft:
    ```text
    AI Strategy for 7–9 figure founders
    Forbes TH AI Thought Leader
    1,000+ students | ฿3,000M+ sales experience
    Limitless Club • Consulting • Training
    ```
    Thai alternative:
    ```text
    AI Strategy สำหรับเจ้าของธุรกิจ 7–9 หลัก
    Forbes TH AI Thought Leader
    ลูกศิษย์ 1,000+ | ประสบการณ์ยอดขาย ฿3,000M+
    Limitless Club • Consulting • Training
    ```
- **Why it increases luxury association:** Luxury brands speak with proof and restraint, not emoji emphasis. Removing the emoji stack makes Jet feel more like a founder-media/institutional advisor and less like a mass-market creator.
- **Quick implementation note:** Prepare 2–3 bio variants for Jet approval; do not edit live profile until approved. Keep one powerful proof number per line and avoid more than one symbol.
- **Priority:** High

### 2) Highlight cover system: replace mixed photo bubbles with charcoal editorial covers
- **Exact change:** Create a matching highlight cover set using `#17181A` charcoal background, warm ivory labels, and rare `#94764A` bronze micro-line/icon. Suggested labels:
  - `START HERE`
  - `AI PLAYBOOK`
  - `TRAINING`
  - `PROOF`
  - `EVENTS`
  - `SERVICES`
  - `FREE`
  - `2026`
- **Why it increases luxury association:** Current highlight bubbles use mixed photos, thumbnails, and one emoji label (`เรียนรู้ฟรี😍`), which creates visual noise. A unified highlight row acts like a hotel concierge menu: calm, useful, expensive.
- **Quick implementation note:** Design square covers first, then crop-test inside Instagram’s circular frame. Keep typography large enough to read at thumbnail size; remove emoji from highlight titles where possible.
- **Priority:** High

### 3) First grid row: make top-row packaging feel like a curated magazine front desk
- **Exact change:** For future pinned/top-row planning, use a deliberate 3-card sequence:
  1. **Limitless Club proof** — dark editorial class/community cover, one hero number, small `Limitless Club` masthead.
  2. **Founder authority** — clean portrait/interview cover with restrained headline and white/ivory type.
  3. **AI insight** — consistent Reel/carousel cover with one clear idea, not multiple stickers or competing objects.
- **Why it increases luxury association:** The current first row has strong proof but mixed visual languages: event collage, portrait/interview frame, and bright podcast-style English text. A curated top row makes the profile feel intentionally directed rather than algorithmically assembled.
- **Quick implementation note:** No need for a rebrand. Start by approving one new cover template for the next 3 uploads, then pin only after Jet approves the exact posts.
- **Priority:** High

### 4) Reel cover footer discipline: add a fixed metadata strip to reduce “AI thumbnail” noise
- **Exact change:** For Reels about tools/agents/co-pilots, standardize a lower footer system:
  - Left: `LIMITLESS CLUB`
  - Center/right: content category, e.g. `AI OPERATING SYSTEM`, `FOUNDER WORKFLOW`, `AGENT PLAYBOOK`
  - Footer CTA: `@jeditrinupab`
  - Limit robot/AI mascot graphics to one supporting element, never the visual hero.
- **Why it increases luxury association:** The visible grid includes several loud AI/robot-style covers with yellow text and app logos. A fixed editorial footer keeps performance-friendly thumbnails while adding brand memory and premium structure.
- **Quick implementation note:** Apply to new Reel covers first. Do not retro-edit older posts unless Jet wants a cleanup sprint.
- **Priority:** Medium

### 5) Typography restraint: reduce mixed headline treatments across carousel covers
- **Exact change:** Use one headline hierarchy for Instagram covers:
  - Small masthead: `LIMITLESS CLUB`
  - Large serif/sans headline: 3–7 words max
  - Small metadata: topic/date/series
  - Footer: `@jeditrinupab`
- **Why it increases luxury association:** Premium media brands repeat typography until the audience remembers the source before reading the handle. The current grid shows multiple treatments: large yellow Thai text, white podcast captions, dark quote cards, event graphics, and AI sticker covers.
- **Quick implementation note:** Build one `Midnight Luxe Editorial` IG cover master file and force every new carousel/Reel cover through it for 7 days.
- **Priority:** Medium

- It is the smallest visible change with the highest first-impression leverage.
- It makes Jet’s existing authority signals feel more expensive without changing the offer.
- It sets the tone for every other profile asset: less emoji, more institution, more founder-media.
