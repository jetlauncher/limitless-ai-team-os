## Micro-upgrades

### 1) Curate the first 5 highlight covers into a premium proof ladder
- **Exact change:** Draft a reordered/pinned first-row highlight sequence: `Start Here`, `AI Workshops`, `Client Wins`, `Corporate`, `Credentials`. Use matching dark charcoal covers, tiny bronze line icon/label, and no emoji in the visible titles.
- **Why it increases luxury association:** Luxury brands make proof easy to navigate and visually calm. A proof ladder turns the profile from “many stories” into a curated executive portfolio.
- **Quick implementation note:** Draft only for Jet approval. Keep existing content inside highlights; change only visible cover/title/order where possible. Use `#17181A`, `#E2D7C8`, rare `#94764A`.
- **Priority:** High

### 2) Replace emoji-led bio lines with an editorial proof stack
- **Exact change:** Draft a calmer bio version: `AI systems for 7–8 figure founders` / `1,000+ students • Corporate workshops` / `Founder, Limitless Club` / `Free AI training ↓`.
- **Why it increases luxury association:** Emojis can feel approachable, but premium first impressions usually rely on restraint, status, and specificity. Proof remains, but the profile feels more expensive.
- **Quick implementation note:** Thai version can preserve warmth: `AI Systems สำหรับเจ้าของธุรกิจ 7–8 หลัก` / `ลูกศิษย์ 1,000+ • Corporate Workshops` / `Founder, Limitless Club` / `เรียนฟรีด้านล่าง ↓`. Draft for approval only; do not edit live.
- **Priority:** High

### 3) Add a quiet footer system to carousel/Reel covers going forward
- **Exact change:** For future cover graphics, add one consistent bottom/footer rule: `LIMITLESS CLUB • @jeditrinupab` in warm ivory, with a thin bronze line. Keep it small and identical across posts.
- **Why it increases luxury association:** Repeated editorial metadata builds brand memory without shouting. It also connects AI tips, corporate proof, and guest clips into one premium media system.
- **Quick implementation note:** Apply only to new covers first; no need to redesign the archive.
- **Priority:** Medium-High

### 4) Make proof/photo posts less “event recap” and more “institutional record”
- **Exact change:** For cohort and corporate training covers, keep the real photo but add a restrained label block: `LIMITLESS CLUB / COHORT 21` or `CORPORATE AI WORKSHOP / TGM`, with date/location in tiny text. Reduce oversized numeric overlays unless the number is the main story.
- **Why it increases luxury association:** High-end institutions document moments like records, not flyers. This keeps social proof while removing promotional loudness.
- **Quick implementation note:** Use a translucent charcoal gradient bottom third, ivory type, and one bronze separator. Start with the next cohort/training post.
- **Priority:** Medium

### 5) Standardize bilingual naming conventions
- **Exact change:** Decide one naming pattern for public labels: either Thai-first with English sublabel, or English-first with Thai captions. Avoid a mixed row like `เรียนรู้ฟรี😍`, `Corp. Training`, `บรรยากาศคลาส`, `My Services`, `Wins!`.
- **Why it increases luxury association:** Consistent naming feels curated and expensive; mixed casual labels feel like an evolving personal archive.
- **Quick implementation note:** Recommended visible highlight labels: `Start Here`, `Free Training`, `Workshops`, `Results`, `Credentials`, `Story`, `Services`.
- **Priority:** Medium
