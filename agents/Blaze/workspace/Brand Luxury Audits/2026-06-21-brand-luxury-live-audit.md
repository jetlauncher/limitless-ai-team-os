# Brand Luxury 1% Live Audit — YouTube — 2026-06-21

## Scope
- Focus surface: YouTube `youtube.com/@jeditrinupab`
- Rotation rationale: latest saved audit was Instagram on 2026-06-19, so today’s priority is YouTube.
- Live access: public logged-out YouTube channel page loaded successfully. Homepage header, banner, avatar, visible bio line, first video row, popular videos, AI playlist row, Shorts row, and Posts row were visually inspected.
- No public assets were edited, posted, scheduled, boosted, purchased, or changed.

## First-impression read
Jet’s YouTube already has credible authority signals: 83.5K subscribers, 1.2K videos, strong portrait avatar, visible social proof in the banner, and high-performing AI course/tool content. The channel feels active and useful. The luxury gap is not credibility; it is packaging discipline.

The current homepage first impression has premium elements — black banner base, Jet portrait, Limitless Club mark, high-production avatar — but mixes them with creator-style urgency: a large red Subscribe button graphic inside the banner, many collage photos, varied thumbnail systems, orange/yellow/blue/red accents, big exclamation marks, and inconsistent cover typography. This makes the channel feel energetic and educational, but slightly less expensive than the Limitless Club “premium founder-media” direction.

## Micro-upgrades

### 1) Replace the banner’s red subscribe graphic with an editorial proof line
- **Exact change:** For Jet approval, update the channel banner so the right-side red `SUBSCRIBE` button graphic becomes a quiet editorial proof block, for example:
  - `LIMITLESS CLUB`
  - `AI Systems for Founders & Operators`
  - `1,000+ students • 83K+ YouTube community`
  Keep the Jet portrait and event/community proof, but reduce the number of small collage images or darken them behind a charcoal overlay.
- **Why it increases luxury association:** Red subscribe graphics are familiar YouTube growth cues, but luxury brands avoid begging for action in the hero image. A restrained proof line makes the channel feel like a founder-media institution, not a standard creator channel.
- **Quick implementation note:** Use `#17181A` background, warm ivory text `#E2D7C8`, and one rare bronze accent `#94764A`. Preserve subscribe conversion through YouTube’s native button below the header.
- **Priority:** High

### 2) Create a thumbnail “luxury spine” for AI tool videos
- **Exact change:** Keep expressive faces and strong hooks, but lock the next 10 thumbnails into one shared system:
  - Charcoal/black base or dark gradient.
  - One large Thai headline, max 3–5 words.
  - One bronze/ivory metadata pill: `AI AGENT`, `CLAUDE`, `HERMES`, `COURSE`, etc.
  - Jet portrait crop in a consistent position.
  - Only one high-saturation alert color per thumbnail, used sparingly.
- **Why it increases luxury association:** The current first row jumps between orange Token, black/yellow AI, blue airplane app, coral AI Clash, and teal/gold Hermes. The topics are relevant, but the row reads like separate campaign ads rather than one premium channel system.
- **Quick implementation note:** Do not redesign old thumbnails immediately. Start prospectively with the next upload and make the first row slowly become more coherent.
- **Priority:** High

### 3) Add a calmer title prefix system for flagship assets
- **Exact change:** Replace bracket-heavy or mixed-format labels over time with an editorial naming system. Draft examples:
  - `[Full Course] วิธีการสร้าง AI Operating System ด้วย Claude Cowork` → `Full Course: สร้าง AI Operating System ด้วย Claude Cowork`
  - `Claude Opus 4.8 vs. ChatGPT 5.5 ใครจะชนะ?` → `AI Clash: Claude Opus 4.8 vs ChatGPT 5.5`
  - `คู่มือติดตั้ง Hermes Agent เสร็จใน 20 นาที` → `Hermes Agent: ติดตั้งให้เสร็จใน 20 นาที`
- **Why it increases luxury association:** Editorial series names feel more premium than ad hoc title structures. They also create brand memory: viewers learn the channel has recurring formats, not random uploads.
- **Quick implementation note:** Apply to future titles first; only update old high-traffic evergreen videos if Jet wants a controlled metadata refresh.
- **Priority:** Medium

### 4) Use homepage shelves as a premium conversion path, not just chronology
- **Exact change:** For Jet approval, consider arranging homepage shelves like:
  1. `Start Here: AI Systems for Business Owners`
  2. `Latest AI Agent Briefings`
  3. `Full Courses`
  4. `Case Studies / Results`
  5. `Shorts`
  The current visible order starts with videos, popular videos, AI playlist, Shorts, Posts.
- **Why it increases luxury association:** Premium channels guide the viewer like an editorial publication. A “Start Here” shelf turns a busy content archive into a curated founder journey.
- **Quick implementation note:** This is a YouTube Studio layout change only after Jet approval. No content needs to be deleted.
- **Priority:** Medium

### 5) Tighten the channel description preview into an authority line before the Line CTA
- **Exact change:** For Jet approval, change the visible first line from a direct training CTA only into a proof + positioning line, then the CTA. Draft:
  ```text
  AI Systems & Growth Strategy for Thai founders — by Jedi Trinupab.
  เช็ครอบอบรมกับทีมผม Line OA 👉 https://lin.ee/4v2Xsum
  ```
- **Why it increases luxury association:** The current preview starts with `เช็ครอบอบรมกับทีมผม Line OA 👉 ...`, which is conversion-focused but not premium-positioning-first. A one-line authority frame makes the viewer understand the channel before being asked to click.
- **Quick implementation note:** Keep the same link and conversion path. Only add the positioning sentence above it.
- **Priority:** Medium

## Today’s recommended 1% upgrade
**Draft a cleaner YouTube banner revision that removes the red subscribe-button graphic and replaces it with a restrained Limitless Club proof line.**

## Why this is the best 1% move
- The banner is the largest visual brand asset above the fold; it sets the price/authority perception before anyone watches a video.
- YouTube already provides a native Subscribe button, so the red graphic duplicates a functional UI element and makes the hero feel more creator-growth than premium founder-media.
- A charcoal/ivory/bronze proof block would align the YouTube first impression with the Midnight Luxe Editorial system without requiring a channel rebrand.
