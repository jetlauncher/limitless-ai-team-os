# Review Gates

The AI Brain should compound automatically, but important changes need review.

## Safe automatic updates

- Create empty placeholder files for obvious people/projects.
- Add backlinks to MOC pages.
- Write daily summaries.
- Add source metadata.
- Draft proposed memory bullets.

## Needs human review

- Anything involving money, legal, health, family, staff performance, or customer-sensitive data.
- Deleting, renaming, or merging source files.
- Publishing, emailing, posting, or deploying.
- Changing credentials, auth, tokens, or billing.
- Promoting a strong opinion into durable memory when evidence is weak.

## Review Queue format

```markdown
# Review Item — <date> — <short title>

## Proposed change

## Source evidence

## Why it matters

## Risk

## Recommended action

- [ ] Approve
- [ ] Edit
- [ ] Reject
```
