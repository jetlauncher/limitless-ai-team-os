# Folder Structure

Use this structure for a student or lightweight company AI brain.

```text
AI Brain/
├── USER.md
├── SOUL.md
├── IDENTITY.md
├── People/
├── Projects/
├── Decisions/
├── Companies/
├── Meetings/
├── Daily/
├── Knowledge/
├── MOC/
├── Sources/
│   ├── YouTube Videos/
│   │   ├── Transcripts/
│   │   └── Summaries/
│   ├── Plaud Transcripts/
│   ├── Brain Dumps/
│   ├── Apple Notes/
│   └── References/
└── Review Queue/
```

## Folder purposes

| Folder | Purpose | What goes here |
|---|---|---|
| `People/` | Memory about important humans | preferences, communication style, relationships, contact pointers |
| `Projects/` | Active outcomes | goals, milestones, status, open loops |
| `Decisions/` | Why choices were made | decision, owner, date, reason, alternatives |
| `Companies/` | External/internal companies | research, competitors, partnership context |
| `Meetings/` | Extracted meeting knowledge | decisions, commitments, preferences, insights |
| `Daily/` | Lightweight operating log | 3–7 lines/day, important changes only |
| `Knowledge/` | Reusable ideas | frameworks, quotes, mental models, playbooks |
| `MOC/` | Maps of Content | index pages that link scattered notes into one concept |
| `Sources/` | Immutable source archive | transcripts, raw notes, imported reference material |
| `Review Queue/` | Human approval | anything risky before durable promotion |
