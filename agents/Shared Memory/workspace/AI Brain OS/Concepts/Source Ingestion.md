# Source Ingestion

Source ingestion is the process of turning videos, transcripts, notes, articles, and voice memos into durable, searchable AI-brain material.

## Pipeline

```text
Capture → Raw archive → Clean transcript → Structured extraction → Durable facts → Links/MOCs → Review
```

## Key rule

Never summarize from title/thumbnail alone. Ingest the real source body or transcript first.
