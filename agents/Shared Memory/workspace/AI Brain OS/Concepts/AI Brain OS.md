# AI Brain OS

An AI Brain OS is a persistent, inspectable context system that lets AI assistants use a human or company memory across sessions.

## Formula

```text
Identity + Source Archive + Structured Memory + Concept Links + Scheduled Compounding + Review Gates
```

## Why it matters

Without persistent context, AI behaves like a smart stranger. With a brain, it becomes an operating partner that remembers preferences, decisions, people, projects, and frameworks.

## Limitless OS implementation

- Identity lives in Hermes profile personas, built-in memory, and agent docs.
- Source archive lives in `Agents/Shared Memory/Sources/`.
- Agent memories live in `Agents/<Agent>/Memory/MEMORY.md`.
- Daily logs live in `Agents/<Agent>/Daily/`.
- Shared handoffs live in `Agents/Shared Memory/Daily/`.
- Qwen can run low-cost hygiene and compounding passes.
