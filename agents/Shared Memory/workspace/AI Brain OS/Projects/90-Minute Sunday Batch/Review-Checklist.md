# Sunday Content Engine — Review Checklist

## Source grounding
- [ ] Every item has a source tag: Plaud / YouTube / Saved X / Watched YouTube / AI Lab Update.
- [ ] Real-news reels are based on actual current news, not generic AI trends.
- [ ] No unverifiable claims are stated as facts.

## Jet voice
- [ ] Sounds like a founder/operator building AI Team OS.
- [ ] Uses business-owner language, not tool-influencer language.
- [ ] Has concrete example, math, or operator consequence.
- [ ] Avoids generic AI hype.

## Format QA
- [ ] Carousel: one idea per slide, strong cover hook, save-worthy lesson.
- [ ] Reel: <=90 seconds, one hook, one point, one CTA.
- [ ] YouTube idea: clear viewer promise, title options, rough outline.
- [ ] LinkedIn: English, founder-level, polished but direct.
- [ ] IG Threads: Thai-native, sharp, practical.
- [ ] X posts: English, concise, high-signal.
- [ ] X threads: strong opening + logical thread arc.

## Publishing readiness
- [ ] CTA matches current offer or intent.
- [ ] Thai/English routing is correct.
- [ ] No sensitive/private student/customer info.
- [ ] No fake screenshots, fake metrics, or fabricated source claims.
