# Sunday Content Engine — 90-Minute Runbook

## 0–10 min — Source selection
Pick:
- 1–3 Plaud transcripts/summaries
- 1–2 YouTube videos/transcripts
- saved X via Readwise MCP if available (`readwise` server; Reader/X-save query or tag required)
- watched YouTube export if available
- latest AI lab/news updates

## 10–25 min — Source digest
Extract ideas, hooks, proof, stories, objections, and news angles.

## 25–40 min — Content architecture
Assign ideas to pillars and output types.

## 40–70 min — Draft generation
Generate all required output types into the output table.

## 70–82 min — Review and revision
Run checklist and revise/reject weak items.

## 82–90 min — Notion packaging
Create/update the master Notion page and subpages.
