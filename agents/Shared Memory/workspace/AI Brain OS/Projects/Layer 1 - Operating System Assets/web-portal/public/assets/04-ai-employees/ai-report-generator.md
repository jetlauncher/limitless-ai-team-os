# AI Report Generator

## Purpose
Convert business data into decision-ready summaries for the owner/team.

## KOI Brief

### Knowledge
- Revenue targets
- Dashboard definitions
- Data sources
- Metric thresholds
- Report format examples

### Output
- Daily/weekly report
- Anomaly list
- Decision memo
- Trend summary
- Next-action recommendations

### Instructions
- Start in draft-only mode.
- Use approved sources only.
- Log every output and escalation.
- Improve the Knowledge Loop when the human corrects an answer.

## Approval gates
- Financial advice
- Changing data
- External reporting
- Forecast claims without caveats

## Best first workflow
Produce a Friday weekly performance report with anomalies and decisions required.

## 10-80-10 usage
First 10%: human defines outcome/source/boundaries. Middle 80%: AI drafts/summarizes/prepares options. Final 10%: human checks judgment and saves corrections.
