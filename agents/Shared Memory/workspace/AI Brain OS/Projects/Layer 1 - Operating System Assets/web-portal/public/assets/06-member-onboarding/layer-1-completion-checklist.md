# Layer 1 Completion Checklist

A member has completed Layer 1 when they have:
- [ ] One AI strategy map using Golden Triangle
- [ ] One Knowledge Loop inventory
- [ ] One KOI role brief
- [ ] One CLEAR prompt used in a real task
- [ ] One 10-80-10 delegation template filled out
- [ ] One AI employee selected
- [ ] One first workflow shipped in draft-only mode
- [ ] One approval boundary map
- [ ] One success metric for the next 14 days
- [ ] One correction saved back into the Knowledge Loop

## Graduation standard
The member is not done when they understand the concepts. They are done when one AI workflow produces a useful artifact that a human can review.
