# Layer 1 Productization Map

## Product layer
Layer 1 = **The Operating System**: software + playbooks that make AI usable inside a real business.

## Asset-to-product mapping

| User need | Asset | Portal feature |
|---|---|---|
| "Where should AI fit in my company?" | Golden Triangle Strategy Map | Interactive strategy worksheet |
| "How do I write better prompts?" | CLEAR Prompt Library | Department prompt browser |
| "How do I delegate without losing quality?" | 10-80-10 Templates | Delegation brief builder |
| "How do I create an AI role?" | KOI Method | AI Employee role generator |
| "Which AI employee do I start with?" | AI Employee Library | Role selector / quiz |
| "How do I implement safely?" | Approval Boundary Map | Safety checklist |
| "How do I get a first win?" | 7-Day Sprint | Guided implementation track |

## MVP portal features
1. Static content library
2. Downloadable worksheets
3. Prompt copy buttons
4. AI Employee role cards
5. Progress checklist
6. First workflow submission form

## Premium portal features later
1. Member workspace per company
2. Saved strategy map
3. AI employee builder that exports a role brief
4. Prompt execution inside portal
5. Approval workflow tracker
6. Weekly report generator
7. Knowledge Loop update log

## Recommended first member journey
1. Watch Start Here
2. Fill Golden Triangle worksheet
3. Pick one AI employee
4. Fill KOI brief
5. Use one CLEAR prompt
6. Run one 10-80-10 delegation
7. Review one output
8. Save one correction into Knowledge Loop

## Sales positioning
Do not sell Layer 1 as "templates". Sell it as the **business operating layer** that turns AI from random prompt usage into repeatable work.
