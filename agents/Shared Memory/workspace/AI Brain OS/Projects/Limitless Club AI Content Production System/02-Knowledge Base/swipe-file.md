# Swipe File

Use this only for **packaging inspiration** after the idea exists.

Never use outliers as the primary source of ideas.

## Title pattern examples

- I Tried X For Y Days — Here Is What Changed
- How I Built X With Only Y
- Why X Is Not The Real Problem
- I Replaced X With AI — The Result Surprised Me
- The 3-Step System I Use To X

## Thumbnail rules

- Max 3–4 words.
- Do not repeat the title.
- Create one curiosity gap with the title.
- One emotion only: shock, relief, fear, ambition, disbelief.
