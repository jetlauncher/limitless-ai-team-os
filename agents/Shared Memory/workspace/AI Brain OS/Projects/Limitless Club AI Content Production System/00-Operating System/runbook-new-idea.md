# Runbook — New Idea Seed

## Step 1 — Create bundle

Copy:

```text
03-Idea Bundles/_template/
```

to:

```text
03-Idea Bundles/YYYY-MM-DD_idea-slug/
```

## Step 2 — Fill `00-brief.md`

Minimum fields:

- Raw idea.
- Source type: Spark / Brain dump / Recording.
- Target audience segment.
- Offer/CTA candidate.
- Proof points / numbers / story.

## Step 3 — Run agents

Use these prompts:

1. `01-Agent Prompts/agent-1-longform-director.md`
2. `01-Agent Prompts/agent-2-shortform-agent.md`
3. `01-Agent Prompts/agent-3-clipping-agent.md`

## Step 4 — Jedi 5-minute review

Jedi chooses:

- final long-form package
- recording format
- 3–5 priority shorts
- A-grade clips/mids to edit first

## Step 5 — Production

Editor/tool execution layer uses:

- long-form edit plan
- short-form editor directives
- clip map timestamps

## Step 6 — Learning loop

After 7 and 28 days, update:

```text
04-Performance Logs/proven-content-log.md
```
