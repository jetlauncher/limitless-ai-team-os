# Short-form Batch

Parent long-form angle: **Before hiring more marketers, build the AI agent workflow first.**

## Script 1 — คุณไม่ได้ขาดทีมการตลาด

**Platform:** IG Reels / TikTok / Shorts  
**P.A.C.T.:** P5 / A4 / C4 / T5 = **18/20**

**Hook 0–3s:**  
"คุณไม่ได้ขาดทีมการตลาด คุณขาดระบบ"

**Context 3–10s:**  
"นักเรียนผมคนหนึ่งคิดว่าต้องจ้างเพิ่ม 3 คน เพราะงาน marketing เริ่มล้น"

**Payload 10–45s:**  
"แต่พอเราแตกงานออกมา จริง ๆ มันมี 3 งานซ้ำ: คิดไอเดียโพสต์, เขียน draft, และจัดคิวงานให้เจ้าของตรวจ พอเราทำ 3 งานนี้เป็น AI workflow เจ้าของไม่ได้เสีย control แต่ทีมทำงานเร็วขึ้นมาก"

**Close:**  
"ก่อนจ้างคนเพิ่ม ลองถามก่อนว่างานนั้นควรเป็นระบบหรือควรเป็นคน"

**Editor directive:** first frame big text `คุณไม่ได้ขาดคน`; cut to workflow screen at 7s; zoom every 3s; gold highlight on `ระบบ`.

## Script 2 — AI ไม่ใช่ Chatbot

**Platform:** IG Reels / Facebook Reels  
**P.A.C.T.:** P5 / A5 / C4 / T4 = **18/20**

**Hook:**  
"ถ้าคุณใช้ AI แค่พิมพ์ถามตอบ คุณยังใช้มันไม่ถึง 10%"

**Payload:**  
"Chatbot ตอบคำถาม แต่ agent รับผิดชอบงานเป็น loop เช่น รับ brief → แตก content → เขียน draft → ส่งให้ตรวจ → ปรับตาม feedback ความต่างคือมันไม่ได้ช่วยคิดครั้งเดียว แต่มันเข้าไปอยู่ใน workflow"

**Close:**  
"ธุรกิจยุคใหม่ไม่ได้ชนะด้วย AI tool เยอะกว่า แต่ชนะด้วย workflow ที่ดีกว่า"

**Editor directive:** show split screen: `Chatbot = answer` vs `Agent = workflow`.

## Script 3 — ก่อนจ้าง Marketer เพิ่ม

**Platform:** TikTok / IG Reels  
**P.A.C.T.:** P5 / A5 / C5 / T4 = **19/20**

**Hook:**  
"ก่อนจ้าง marketer เพิ่ม ให้ถาม 3 ข้อนี้"

**Payload:**  
"หนึ่ง งานนี้ซ้ำทุกสัปดาห์ไหม สอง ต้องใช้ taste ของเจ้าของจริง ๆ หรือแค่ต้องใช้ข้อมูลเดิม สาม ถ้า AI ทำ draft 70% แล้วคนตรวจ 30% ได้ไหม ถ้าตอบว่าใช่ งานนี้ควรเป็น workflow ก่อนเป็น headcount"

**Close:**  
"คนเก่งยังสำคัญ แต่ให้คนเก่งทำงานที่ AI ทำแทนไม่ได้"

**Editor directive:** numbered text cards, 1 question per screen, fast cuts.

## Script 4 — ทีม 3 คน ใน Agent เดียว

**Platform:** YouTube Shorts / IG Reels  
**P.A.C.T.:** P4 / A4 / C4 / T5 = **17/20**

**Hook:**  
"ผมไม่ได้ให้ AI แทนคน ผมให้ AI แทนงานซ้ำ"

**Payload:**  
"ใน marketing team มี strategist, copywriter, coordinator แต่ในธุรกิจเล็ก หลายงานยังเป็น pattern เดิม เราเลยให้ AI agent ทำ draft และ coordination ก่อน แล้วให้เจ้าของตัดสินใจเรื่อง brand, offer, และ final approval"

**Close:**  
"นี่คือวิธีคิดแบบ owner ที่อยาก scale โดยไม่เพิ่ม chaos"

**Editor directive:** overlay org chart: human team → agent workflow → owner approval.
