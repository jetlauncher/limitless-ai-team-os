## Script 56 — "ภาษีที่แพงที่สุดคือความไม่รู้" 🔀
**Hook:** "ภาษีที่แพงที่สุดในชีวิตคุณ ไม่ใช่ภาษีเงินได้"
**Setup:** มันคือภาษีความไม่รู้ Ignorance Tax ที่เราจ่ายเพราะไม่รู้ว่ามีสิ่งนี้อยู่
**Payoff:** 8 ปีก่อน ผมเสียเงินไปกว่า 1,500,000 บาท เพราะความไม่รู้ และเจ็บกว่านั้นคือเวลา โอกาส และพลังใจที่หายไป
**Close:** ความไม่รู้ มีราคาเสมอ

## Script 60 — "เปลี่ยนร่างกาย เปลี่ยนทุกอย่าง" 🔀
**Hook:** "ผมควบคุมสิ่งที่เกิดกับผมไม่ได้ แต่ควบคุมร่างกายผมได้"
**Setup:** 16 ปีก่อน ทุกอย่างพังทลาย ความสัมพันธ์ 5 ปีจบลง ผมได้ยินเสียงในหัวว่า "ทำไม่ได้หรอก"
**Payoff:** ผมตัดสินใจเปลี่ยนร่างกายให้สมบูรณ์ ใน 2 ปี ผมมี 6 pack, bodyfat 6-8% และเมื่อพิสูจน์ได้ว่าเปลี่ยนได้ในด้านหนึ่ง ทุกด้านก็เป็นไปได้
**Close:** ชนะที่ร่างกาย ชนะทั้งชีวิต
