## How to use this
- Each script = **Hook (0-2s) → Setup (2-10s) → Payoff (10-30s) → Close (last 3s)**
- Film **walking or in motion**, natural light, one idea per clip, 15-45s
- Caption = the hook text + 3 hashtags, nothing more
- No "follow for more." End quiet and confident.
- Film 5 per session. Post 1/day. Cross-post IG → TikTok → YT Shorts.

## Language key
- 🇹🇭 = deliver in Thai (higher comments/engagement, Thai-first audience)
- 🌎 = deliver in English (higher reach/saves/shares)
- 🔀 = Thai with English power-words mixed in (your natural style)
