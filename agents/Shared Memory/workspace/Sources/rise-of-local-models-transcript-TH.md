## ทรานสคริปต์เต็ม (ไทย)

เพื่อนๆครับ ความฝันที่พวกเราจะมี Super Intelligence โดยเฉพาะพวกเราคนไทย ที่เป็นผู้ประกอบการที่อยู่ในประเทศที่ถือว่าค่อนข้างจะเล็ก ถ้าเทียบกับผู้นำและนักธุรกิจที่อยู่ในสหรัฐอเมริกา เรียกว่าเพิ่งแบบดิ้นหายไปเรียบร้อยเลยครับ เหมือนกับโดนธานอสดีดแล้วก็หายไป

สิ่งที่เกิดขึ้น ถ้าใครได้ติดตามข่าว ก็จะเห็นว่าล่าสุด นอกเหนือจากที่ Anthropic ได้ปล่อยตัว model ที่ฉลาดที่สุดของเขาออกมา และเพียงแค่ไม่กี่วัน รัฐบาลสหรัฐก็บอกว่า Anthropic ต้องหยุดให้บริการตัว model นี้สำหรับทุกคนที่เป็นชาวต่างชาติ รวมถึงประเทศที่อยู่นอกสหรัฐอเมริกา และรวมถึงห้ามไม่ให้พนักงานของประเทศต่างๆ ที่ไม่ใช่คนอเมริกา ใช้และพัฒนา model เหล่านี้ด้วยเช่นกัน

ตอนแรก ผมก็คิดว่า เออ เดี๋ยว OpenAI ก็น่าจะออกมาเป็นสวิงคีย์มากขาวมากนะครับ "Hey I got you guys เราเป็นป้าใจดี เราจะให้ทุกคนใช้ตัว model ที่เก่งที่สุดของโลก แล้วก็เปิดให้ทุกคนได้ใช้พร้อมๆกัน" เพราะมันน่าจะเป็นการตัดสินใจที่ดี เพราะในขณะที่ Anthropic ปิด ตัว OpenAI เปิด ก็น่าจะทำให้คนอยากจะเปลี่ยนมาใช้มากขึ้นหรือเปล่า

แต่ไม่ครับ เพราะว่ามันมี post บน Twitter ที่ผมอ่านแล้วผมก็แบบ อะไรวะ งงมาก คือ งงแล้วก็ upset มากๆ นะครับ

เขาบอกว่า เปิดตัวว่ามี model ใหม่ GPT 5.6 มา แต่พวกเราใช้ไม่ได้ครับ จะประกาศทำไม? ก็เงียบๆไปดิ ไม่เห็นต้องประกาศเลย แล้วเขาก็บอกว่า "We believe in broad access and plan to make GPT 5.6 Terra and Luna generally available in the upcoming weeks. For now, at the request of the US government..." เน้นย้ำนะครับ เห็นไหมครับ "at the request of the US government, we're starting with a limited preview among a small group of trusted partners in Codex and the API"

small group of trusted partners — น่าเสียดายครับ ผมและคุณไม่ใช่ small group นั้นที่เขาจะไว้วางใจเพียงพอที่จะให้พวกเราได้ใช้งานมัน

ซึ่งมันแปลว่าอะไรครับ สิ่งที่กำลังเกิดขึ้นก็คือ มันจะมีกลุ่มชนชั้นบนสุด ที่ติดกับสหรัฐอเมริกามากที่สุด ที่เป็นคนที่รวยที่สุด เป็นองค์กรที่เคยหรือว่าสนิทสนม ที่เข้าถึง Frontier Model ก่อนผมและคุณ ซึ่งอันนี้มันค่อนข้างจะแย่มากๆ

เพราะมันแปลว่าอะไรครับ ในขณะที่มีองค์กรกลุ่มเล็กๆ ที่ได้เข้าถึง Frontier Model ก่อน คนอย่างพวกเรายังได้อันที่มันฉลาดน้อยกว่า ถึงแม้อันที่มันฉลาดอยู่แล้ว อย่างตัว GPT 5.5 บวกตัว Opus 4.8 บวกกับตัว Gemini พวกนี้ก็เก่งมากๆ แต่ลองคิดดูสิ มันเก่งกว่า

ดังนั้นมันแปลว่าอะไรครับ กลุ่มนี้จะสามารถพัฒนา software ได้เร็วกว่า แก้ bug ได้เร็วกว่า ปิดช่องโหว่ด้านความปลอดภัยได้เร็วกว่า รวมถึงสามารถ launch สร้าง product ที่พวกเราอาจจะไม่เคยคิดฝันว่าจะสร้างได้เลย

## Mac Mini / Mac Studio กับราคาที่พุ่ง

สิ่งที่ผมคาดการณ์ก็คือ ตอนนี้พวก Mac Mini พวก Mac Studio ถ้าใครพยายามจะไปหาซื้อในช่วง 3-4 เดือนที่ผ่านมา ก็จะเห็นว่ามันเกลี้ยงตลาดทั้งหมดเลย พอมีของ ก็โดนซื้อไปในแบบพริบตาเลย

เพราะอะไรครับ เพราะทุกคนไหวตัวทัน เพราะทุกวันนี้ทุกคนกำลังไล่ล่าหา RAM หา GPU หาคอมพิวเตอร์ที่จะมา run AI Agent ของเขาในขณะที่เขาอยู่ข้างนอก ซึ่งไม่ว่าคุณจะใช้ Hermes ใช้ OpenClaw ใช้ Perplexity Comet หรือใช้ Codex หรือใช้ Cloud Work ทุกคนสามารถมี AI Agent ที่อยู่ที่บ้าน เสียบ Internet ไว้ แล้วคุณสามารถสั่งงานได้ 24 ชั่วโมง x7 ในขณะที่คุณอยู่ข้างนอก

ด้วยเหตุนี้มันเลยทำให้ Mac Mini, Mac Studio และคอมพวกนี้ที่เอาไว้ใช้ run AI Agent นั้นเกลี้ยงตลาด แล้วผมเชื่อว่าจากเหตุการณ์ที่เกิดขึ้น หลายๆคนจะเริ่มเห็นแล้วว่า การมี Intelligence หรือมี Local Model ที่เป็น AI ของเราเองในบ้านของเรานั้นสำคัญขนาดไหน

ปัจจุบันผมดีใจมากๆเลยที่ผมกัดฟันซื้อทั้ง Mac Studio ทั้ง MacBook Pro ของผมที่กดไปประมาณ 200,000 นิดๆ ตอนนี้แบบ 200,000 กว่า เพิ่มมาอีก 40,000-50,000 เลย เราเห็นความแตกต่างชัดเจน MacBook Pro M5 ตัว 14 นิ้ว จากเดิม 56,900 บาท ตอนนี้โดดขึ้นมาเกือบจะ 70,000 บาทแล้ว

แล้วผมเชื่อว่า trend นี้จะไม่หยุดแค่นี้ แล้วมันจะแพงขึ้นเร็วมากๆ จากที่ผมดู Mac ผมไม่เคยเห็น Mac ลดราคา ไม่เคยเห็น Mac ดรอปราคา เว้นแต่เป็นเครื่องที่ตกรุ่น แล้วผมเชื่อว่ารุ่นใหม่ๆที่มี chips ที่แรงขึ้น RAM ที่เยอะขึ้น ที่สามารถ run Local AI ได้มากขึ้น ก็จะแพงขึ้นๆ แล้วอีกหน่อยมันจะจับต้องไม่ได้แล้ว

รอบนี้ถ้า M5 Mac Studio ออกมามี RAM สูงเท่าไหร่ ผมก็จะอัด RAM ให้สุด แล้วก็จะซื้อทันทีเท่าที่มันมี เพราะผมเชื่อมากๆว่ามันจะมีเทรนด์แบบนี้เกิดขึ้น และนี่เพียงแค่จุดเริ่มต้น

เพราะทุกวันนี้คนที่ adopt ใช้ AI อย่างแท้จริง คือยังแบบเสี้ยวหนึ่งของโลกเองนะครับ หลายๆคนอาจจะคิดว่าคนใช้เยอะ แต่ไม่คิดมาก่อนเลยว่าแกปมันจะห่างขนาดนี้ ดังนั้นถ้าทุกคนดูช่องนี้อยู่ ผมบอกไว้เลยว่าคุณ at the frontier age แน่นอน แล้วถ้าคุณเอาสิ่งที่ผมสอนไปลงมือใช้ โดยเฉพาะถ้าคุณเป็นลูกศิษย์ผมอยู่แล้ว คือคุณ ahead กับคนส่วนมากทั่วโลกเลย ดังนั้นภูมิใจในตัวเองไว้นะครับ

## Local Model ที่ผมใช้ — Qwen 3.6

ง่ายๆคืออันนี้มันจะโหลดโมเดลเข้าไปใน RAM ดังนั้นถ้า RAM ของคุณน้อยกว่า 16GB หรือน้อยกว่า 22GB คือไม่สามารถใช้ Qwen 3.5 อันนี้ได้ จริงๆตอนนี้ผมใช้เป็นตัว 3.6 แล้ว เดี๋ยวต้อง update ในเครื่องนี้ พอดีจะใช้ผ่าน MacBook Pro แต่ anyways มันอยู่ประมาณ 22GB

เพื่อนๆก็คงสงสัยว่า Qwen 3.6 นี่มันเก่งยังไงบ้าง ผมจะบอกให้ มันเก่งทุกอย่างครับ เก่ง coding เก่งเขียน เก่ง brainstorm เก่งทุกอย่างที่คุณคิดว่ามันควรจะเก่ง เหมือนกับคุณใช้ ChatGPT ประมาณ 5.0 เมื่อปีที่แล้ว Qwen 3.6 เก่งประมาณนั้นเลย

ที่เลือกตอนนี้เพราะว่ามันมีการผสมผสานของความฉลาด และการกิน RAM พอสมควร ทำให้ Mac Studio เครื่องนี้ที่มี RAM 96GB ผมสามารถ load Qwen 3.6 เข้าไป และก็ยังสามารถมี room หรือมี overhead เหลือให้งานอื่นๆของ AI Agent ผมยังทำงานได้ ซึ่งแน่นอนถ้าผมมี RAM แค่ 32GB ผม load Qwen 3.6 เข้าไป 20 กว่า GIG เหลืออีกแค่ 12 GIG ระหว่างอัดวิดีโอแบบนี้ คอมผมอาจจะแฮงค์ได้ ยิ่งพอมาใช้เครื่องนี้ โอ้โห ลื่นเลยครับ ใช้ Qwen เร็วมากๆเลย

## Local Model ตัวใหม่ — GLM 5.2 (Z.AI)

อีกอันหนึ่งที่น่าสนใจก็คือ ตอนนี้มี Local Model ตัวใหม่ออกมาจากค่ายที่ชื่อว่า Z.AI ซึ่ง Z.AI ได้ปล่อยตัวที่ชื่อว่า GLM 5.2 เป็น Open Source ออกมาด้วย

Unfortunately ผมยังไม่สามารถใช้ GLM 5.2 บนคอมของผมได้ เพราะ RAM ยังไม่พอ อันนี้ต้องใช้ RAM ขั้นต่ำประมาณ 200 กว่า GIG ง่ายๆคือต้องมี Mac Studio ที่มี RAM 512 GIG เท่านั้น ถึงจะ run GLM 5.2 ได้ ผมทดลองใช้ GLM 5.2 ในแบบ Cloud version ผ่าน platform ที่ชื่อว่า OpenRouter จากที่ผมทดลองมันเก่งมากๆ กล้าบอกได้เลยว่าเก่งพอๆกับ Opus 4.8 ในการเขียนโค้ดเลย

ที่สำคัญครับ ถ้าเรามาดูราคาของ GLM 5.2 — Input Token เพียงแค่ 0.95 (เหรียญ) แล้ว Output Token อยู่ที่ 3 ถ้าเทียบกับ Cloud Opus 4.8 — Input Token 5 เหรียญ, Output Token 25 เหรียญ มันต่างกันเกือบจะ 10 เท่า แล้ว Input Token ต่างกันประมาณ 5 เท่า

ซึ่งมันแปลว่าอะไรครับ ถ้าคุณเป็นเจ้าของธุรกิจ และคุณมีองค์กรใหญ่ที่คุณกำลังค่อยๆ roll out AI ไปให้กับคนทั้งทีม หรือคุณอยากจะซื้อตัว model ที่จะเอาไปใส่ใน application ของคุณ ถ้าคุณรู้ว่ามี AI ที่เก่งและฉลาดพอๆกับ Opus 4.8 ในราคาที่ถูกกว่าเกือบ 20 เท่า คุณจะเลือกอะไรครับ คุณก็ต้องเลือกตัว GLM สิครับ

ตอนนี้มี application หลายๆตัวที่เขามี AI เป็น core หลักของการสร้างคุณค่าให้กับลูกค้าของเขา หลายๆเจ้าเริ่มเปลี่ยนมาใช้ GLM 5.2 หลายๆเจ้าเริ่มเปลี่ยนมาใช้ model อย่าง Kimi K2.6 หลายๆเจ้าเริ่มเปลี่ยนมาใช้ DeepSeek V4 ที่หลายคนบอกว่าเก่งเท่าๆกับ Opus 4.8 ยิ่งโดยเฉพาะในการทำงานที่เป็น AI Agent แล้วผมเชื่อว่า trend นี้จะมีแต่โตขึ้นๆ แล้วเราจะเห็น Local Model ที่เก่งขึ้นๆ

## Gemma (Google) และ Nemotron (NVIDIA)

อย่างค่าย Google เองก็มีตัว Local Model ที่ชื่อ Gemma ถ้าเพื่อนๆอยากลองเล่นพวก Local Model ก็แนะนำโหลด platform ที่ชื่อ LM Studio มา run บน Mac ของคุณได้เลย พอ download มา install เสร็จ ก็กดดูพวก model ที่คุณสามารถโหลดมาใช้ได้ แล้วคุณก็จะดูได้เลยว่า model ไหนที่คุณสามารถ run บนคอมของคุณได้ตาม spec คอมของคุณ

[⚠️ ตรวจสอบ — โมเดลสะดุดช่วงนี้] อย่างของ Google Gemma มีตัวที่เล็กสุดตั้งแต่ 7.56GB ที่เราสามารถปล่อยลงในเครื่องได้ [...] เพราะว่าเก่งเหมือนกับ Gemini ประมาณ 3.1 หรือประมาณ 2.5 Pro เลย

และก็มีของ NVIDIA — Local Model คือ Nemotron 3 Nano Omni ซึ่งมีขนาดประมาณ 26GB (บางเวอร์ชัน 3.1GB หรือ 2.5GB) แล้วก็สามารถเอามา run บนเครื่องของเราได้ โดยการใช้ RAM เพียงแค่ 26GB ถ้าใครซื้อ MacBook Pro ตัวใหม่ที่กด RAM ไป 128GB แล้วก็คุณลองได้แน่นอน ทดลองได้เลยว่าอันไหนดีที่สุดกับการใช้งานของคุณ แล้วลองดูว่าคุณชอบและถูกจริตกับตัวไหนมากกว่า

## สรุป — The Rise of Local AI

นี่แหละครับ The Rise of Local AI ถ้าอย่างนั้นสิ่งที่คุณสามารถทำได้ในยุคนี้ ถ้าคุณเป็นเจ้าของธุรกิจ และคุณอยากมี Intelligence อยู่ในมือของคุณ ผมยังเชื่อว่าตัว Local Model จะมีตัวที่ดีๆขึ้น แล้วค่า Hardware ก็จะแพงขึ้นๆ

ถ้าผมเป็นคุณ คุณเข้าใจหลักการนี้แล้ว และคุณมี use case ชัดเจนว่าจะเอาตัว Local AI ไปใช้อะไร เพราะเมื่อคุณมี Local AI แปลว่าคุณสามารถมี GLM 5.2 เขียน Code, Debug, Audit ให้คุณได้ 24 ชั่วโมง และคุณไม่ต้องจ่ายค่า Cloud สักบาท คุณสามารถปล่อยมัน run สร้างงานใหม่ๆที่คุณไม่เคยสร้างได้มาก่อน เพราะติดเรื่อง usage ติดเรื่องค่าใช้จ่าย แต่ด้วยตัว Local Model มันเหมือนกับเปิดกล่อง Pandora ให้คุณสามารถทำงานต่างๆที่คุณอาจไม่เคยคิดฝันมาก่อน

แล้วถ้าคุณยังคิดไม่ออกว่าต้องทำอะไร คุณลองถาม AI Agent ที่คุณใช้ดูก็ได้ครับ ลองถาม ChatGPT ลองใช้ Claude หรือลองใช้ Gemini ดูก็ได้ว่า ถ้าคุณมีตัว Local Model งานอะไรที่คุณสามารถทำได้ แล้วลองไปดูซิว่ามันคุ้มค่าที่คุณจะ invest ใน hardware หรือไม่

ส่วนตัวผมคุ้มแน่นอนครับ เพราะผมมี use case ต่างๆมากมาย ผมสามารถรับทำงานให้กับลูกค้าได้มากขึ้น หาข่าวที่ดีได้มากขึ้น upgrade ความรู้ของผมได้มากขึ้น เข้าถึง idea ใหม่ๆได้มากขึ้น ผ่าน Local Model แบบนี้ที่ run ให้ผม 24 ชั่วโมง x7 รวมถึงเวลามี Local Model บน MacBook Pro ของผม ผมสามารถอยู่แบบเครื่องปิด (offline) แล้วผมก็ยังทำงานได้ โดยไม่ต้องพึ่ง Internet เลย นี่คือเทคโนโลยีที่เรามี อยู่ในมือของพวกเราแล้ว ณ จุดวันนี้นะครับ

สุดท้าย มันคือ ไม่ว่า AI จะเก่งขนาดไหน มันก็ขึ้นอยู่กับว่าเราหยิบมาใช้เป็นไหม แล้วถ้าคุณดูมาถึงตรงนี้ แล้วคุณยังไม่แน่ใจว่าจะเอา AI ไปใช้ในองค์กรยังไง เดี๋ยวคลิปต่อไป จะแนะนำให้คุณไปดูคลิปที่ผมแนะนำวิธีการสร้างระบบ AI Operating System ไว้เรียบร้อยแล้ว รวมถึงคู่มือ AI Agent 101 ให้กับคนที่ยังไม่แน่ใจจะเริ่มตรงไหน
