# AI Agent Command Center

Use this section in Discord to order work from the whole agent team.

## Fast syntax

```text
Kelly, route this to <agent>: <task>. Deliver here in Discord.
```

Examples:

```text
Kelly, route this to Signal: scan today's most important AI agent news and give me the top 5.
Kelly, route this to Blaze: turn this idea into 3 LinkedIn post drafts in Jet voice.
Kelly, route this to Bolt: build a quick landing page mockup for this offer and send preview files here.
```

## Default delivery rule

Unless Jet says otherwise:
- Work can be ordered from Telegram or Discord.
- Team-facing results deliver to this Discord command channel.
- Private/family agents stay outside team Discord unless explicitly approved.

## If unsure

Use:

```text
Kelly, decide the right agent and run this: <task>. Deliver here.
```
