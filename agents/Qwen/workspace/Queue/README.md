# Qwen Queue

Drop local/background tasks here for Qwen.

## Task lifecycle
- `todo/` — tasks Qwen may process
- `doing/` — active work
- `done/` — completed tasks and outputs
- `archive/` — old completed tasks

## Good task types
- Summarize this folder/file/session
- Cluster these research notes
- Extract content hooks from this transcript
- Review logs and list likely issues
- Clean up duplicate/stale Obsidian notes
- Draft a briefing for Kelly to review

## Bad task types
- Send/post/email anything
- Delete important files
- Deploy production changes
- Decide payment/revenue alerts
- Finalize customer-facing copy

Use `_task-template.md` for new tasks.
