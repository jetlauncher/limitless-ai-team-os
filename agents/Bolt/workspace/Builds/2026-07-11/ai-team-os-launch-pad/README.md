# AI Team OS Launch Pad v0

A single-file HTML launch card for Jet and students to route AI Team OS work quickly.

## Open
- `index.html`

## What it includes
- 60-second workflow: outcome → owner agent → memory handoff
- Agent routing map
- Printable daily checklist
- Student copy-paste prompt

## Verification
- `index.html` is standalone and contains the full HTML/CSS structure.
- No network calls, no external APIs, no deploys.
