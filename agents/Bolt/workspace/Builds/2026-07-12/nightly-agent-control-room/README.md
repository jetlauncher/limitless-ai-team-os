# Nightly Agent Control Room v0

A safe local-only dashboard generated by Kelly's nightly 2:00 AM Bangkok memory sync cron.

## What it does
- Reads recent daily notes from the canonical Obsidian Agents workspace.
- Shows each active/present agent's latest daily-note status.
- Flags local blocker phrases such as `Needs Kelly review`, `credits depleted`, `failed`, and incomplete sync markers.
- Includes recent shared handoffs so Jet can quickly see what changed.
