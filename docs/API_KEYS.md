# API Key Checklist

## Core model providers

- OpenRouter: `OPENROUTER_API_KEY`
- OpenAI: `OPENAI_API_KEY`
- Anthropic: `ANTHROPIC_API_KEY`
- Google Gemini: `GOOGLE_API_KEY` or `GEMINI_API_KEY`

## Messaging

- Telegram BotFather: one `TELEGRAM_BOT_TOKEN` per agent profile.

## Productivity / business systems

- Notion: `~/.config/notion/api_key`
- Beehiiv: `~/.config/beehiiv/api_key`
- Airtable: `~/.config/airtable/api_key`
- Google Workspace: OAuth token generated by your Google setup flow.
- GitHub: `gh auth login` or `GITHUB_TOKEN`

## Creative / media

- OpenAI Images: `~/.config/openai/api_key`
- Higgsfield MCP: `HIGGSFIELD_MCP_TOKEN`
- Blotato: `~/.config/blotato/api_key`

## Local AI

- Ollama server at `http://127.0.0.1:11434/v1`
- Example model: `qwen3.6:35b`

Never paste real keys into this repo.
